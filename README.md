# Smart Learning App 🚀

This is a Python CLI application I built while learning intermediate Python.  
The goal was to combine multiple small projects into one structured app and improve my problem-solving and coding skills.

## 🔧 Features
- 🧠 Math Quiz – practice multiplication with scoring and time tracking  
- 🧮 Calculator – perform basic operations and convert temperature  
- 🎮 Trading Game – a simple simulation game based on randomness and betting  

## 💡 What I Learned
- Writing clean and modular Python code  
- Handling user input and errors effectively  
- Designing logic for interactive programs  
- Organizing multiple features into a single application
